# Przykład konfiguracji

```
1. Założyć projekt w github - sh
2. Zainstalować GIT for windows
3. Zainstalować python for windows
4. Zainstalować vs code
5. Komendy git oraz python3 powinny działać w cmd
6. Pobrać źródła -> git clone https://github.com/jakubgasiorski/hs.git
7. cd hs i komenda -> code .
8. ctrl+j i są komendy w vsc
9. cd server i stworzyć venv -> python3 -m venv venv
10. Zamknąc vsc i otworzyć w katalogu server -> code .
11. stworzyć main.py i vsc przełąduje sobie wszystko
12. stworzyć plik -> requirments.txt i tam dodać bottle
```
